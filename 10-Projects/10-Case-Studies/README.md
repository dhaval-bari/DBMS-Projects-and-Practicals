# SQL Case Studies (10 Sets)

This section contains 10 real-world database case studies covering schema design, SQL queries, and ER diagrams.
These case studies are based on practical scenarios such as e-commerce, hospital systems, airline booking, and more.

Each case study includes:
- Database schema
- 25 SQL queries (SELECT, JOIN, GROUP BY, HAVING, Subqueries)
- ER diagram (via reverse engineering)
