# 🏦 Banking System Database (Besant Bank)

## Scenario

This project simulates a banking system managing accounts and transactions.

---

## Tables

* Accountdetails
* Transactiondetails

---

## Features Implemented

* Account creation and transaction tracking
* Credit/Debit operations
* Views (updatable & non-updatable)
* Stored Procedures (Mini Statement)
* Indexing
* CTE (Common Table Expression)
* Wildcards & Filtering
* Error Handling
* Balance Updates using Transactions
* Account Status (ACTIVE / INACTIVE)
* Cascade Delete (FK constraint)
* Cursor-based processing

---

## Key Concepts Used

* Transactions (COMMIT / ROLLBACK)
* Subqueries
* Views
* Stored Procedures
* Indexes
* CTE
* Joins & Aggregation
